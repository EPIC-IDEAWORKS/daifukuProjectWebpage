Project Scope

TWO PHASES.
DELIVERED.

01
Phase 1 — Infrastructure

Connect the Floor

  • RFID scanners deployed at every cell — Brakepress, Fiber Laser, CO2 Laser, Assembly, Layout 1, H22A, Machine Shop, Door 6 and more
  • Machine-state sensors installed and cloud-connected per cell
  • Stacklights wired in — colour state streamed to AWS in real time
  • MQTT pipeline: sensor events → AWS IoT Core → data store → clean API
  • Live Daifuku dashboard: RFID Overview, Smart Cell view, Plant Details
02
Phase 2 — Intelligence & Automation

Metrics, Insights & Robotics

Cloud Analytics
  • Metrics & trending pipeline — job counts by station, daily flow analysis, threshold monitoring
  • Z-Score anomaly detection for duration of stay at each cell
  • Overall Reporting dashboard — completions/day, capacity gap, days-to-clear, net jobs and median stay
  • Jobs-completed charts with selectable date range; pie chart by plant
COBOT & Hardware
  • Standardized scripting for generic shape operations — simplified COBOT programming
  • Camera investigation: semi-automatic upload of files and schematics to COBOT
  • Barcode scanner integration — auto-scan work orders prior to job execution, capture and log operations
Architecture

DATA,
ALL THE WAY
TO THE CLOUD

Each work order is read passively as it enters a cell. That event is published via MQTT to AWS IoT Core, stored, cleaned, and surfaced in the Daifuku dashboard — within seconds of the physical read.

RFID Reader

Passive tag read at cell entry · sub-100ms · no manual scan

MQTT Publish

Work order event + machine state → AWS IoT Core

Store & Clean

S3 + Lambda pipeline · structured, queryable, analyzed
